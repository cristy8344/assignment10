Enjoy this dog fact!

 
  • I chose this API because I felt it was an interesting dog fact of the day and, well, I just appreciate dogs in general.
  • I found the API through a random google search, https://awesomeopensource.com/project/DukeNgn/Dog-facts-API?categoryPage=1, and the API data can be found here https://dog-facts-api.herokuapp.com/api/v1/resources/dogs/all
  • I used new XMLHttpRequest() to access this data, which allowed me to use open() to request the data from the server (it returns a JSON string). I then implement an event handler to display the data when the data was ready. I parsed the JSON string back into an object and accessed the "text" field.
  • This API updates randomly, so it could be used on a common franchise like Petcos' or PetSmarts' website to generate the animal they cater to.It could also be used on a positive message to include various types of breeds.
